### 1. ¿Cuándo se bloquea una Clave Tributaria?

ID: 001.100.1394.008  
Fecha de creación: 18/05/2004  
Fecha de actualización: 11/02/2025  
URL: https://www.sii.cl/preguntas_frecuentes/clave_secr_repr_electr/001_100_1394.htm

#### Respuesta

La Clave Tributaria se bloquea cuando el usuario realiza múltiples intentos con una clave errónea. Esta medida persigue aumentar el nivel de seguridad, con el fin de proteger al contribuyente del uso indebido que puede realizar un tercero no autorizado por él.

### 2. ¿Qué debo hacer para desbloquear la Clave Tributaria?

ID: 001.100.7418.005  
Fecha de creación: 12/12/2019  
Fecha de actualización: 11/02/2025  
URL: https://www.sii.cl/preguntas_frecuentes/clave_secr_repr_electr/001_100_7418.htm

#### Respuesta

Debe esperar que pase el tiempo de bloqueo (aproximadamente 1 hora). Cumplido el tiempo de bloqueo, existe un intento para ingresar correctamente Clave Tributaria, si vuelve a fallar, hay que esperar nuevamente que pase el tiempo de bloqueo.Si no puede desbloquearla, es posible acceder a la opción Recuperar Clave.

---

### 3. ¿Cuándo se bloquea la Clave Tributaria de un contribuyente fallecido?

ID: 001.100.7419.005  
Fecha de creación: 12/12/2019  
Fecha de actualización: 01/07/2025  
URL: https://www.sii.cl/preguntas_frecuentes/clave_secr_repr_electr/001_100_7419.htm

#### Respuesta

Los herederos podrán usar el RUT y clave del contribuyente fallecido hasta mayo del cuarto año de defunción (contado como primer año el de su fallecimiento), así podrán declarar sin problemas la renta del tercer año. Después de ese tiempo la clave se bloquea para siempre y desde ese momento el usuario ya no puede autenticarse, cambiar ni recuperar la Clave Tributaria del causante.
