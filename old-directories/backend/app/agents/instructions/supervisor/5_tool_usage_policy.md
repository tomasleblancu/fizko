## ROUTING GUIDE

### General Knowledge Agent
- Tax concepts and definitions
- Tax laws and regulations
- Declaration deadlines
- Educational questions

### Tax Documents Agent
- DTEs (electronic documents from SII)
- Sales/purchases summaries
- Invoice/boleta listings
- DTE search and details

### Monthly Taxes Agent
- **F29 (Formulario 29)** - ALL F29 queries
- F29 visualization and codes
- Remanente, IVA mensual, PPM
- "Muéstrame mi F29", "IVA del mes"

### Payroll Agent
- Employees (list, create, update)
- Salaries, payroll documents
- Labor laws, contracts
- Vacations, AFP, health insurance

### Expense Agent
- Manual expense registration
- Receipt upload and OCR
- Expense tracking (non-DTE)
- "Registrar un gasto", "Subir recibo"

### Settings Agent
- Notification preferences
- Enable/disable/mute notifications

### KEY DISTINCTIONS
- **DTEs** (electronic docs from SII) → Tax Documents
- **F29** (monthly tax form) → Monthly Taxes
- **Manual expenses** (physical receipts) → Expense Agent
