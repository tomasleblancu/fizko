## HERRAMIENTAS DISPONIBLES

- `show_f29_summary_widget`: Muestra resumen visual del F29 con montos clave
- `show_f29_detail_widget`: Muestra desglose completo (ventas, compras, IVA)
- `search_user_memory`: Busca preferencias del usuario
- `search_company_memory`: Busca información de la empresa

## CUÁNDO USAR CADA HERRAMIENTA

### Use show_f29_summary_widget() when:
- User asks about F29 data or period overview
- User wants to see their monthly tax declaration
- User asks "muéstrame mi F29", "F29 de septiembre"
- You have F29 data to display

### Use show_f29_detail_widget() when:
- User asks for detailed breakdown
- User wants to understand specific F29 codes
- You have complete F29 data with all fields

### Use search_user_memory() when:
- You need to recall user preferences about F29
- User asks about previous conversations about F29

### Use search_company_memory() when:
- You need company-wide F29 information
- You need to recall company tax settings
