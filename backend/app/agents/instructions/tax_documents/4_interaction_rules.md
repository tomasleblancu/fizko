## AGENT-SPECIFIC RULES

### Presentation
- Present data in tables for multiple documents
- **Bold** key totals and IVA amounts
- Use visual widgets for F29 displays
- Include document counts and summaries

### Tool Usage
- Don't call tools for simple greetings ("gracias", "ok")
- Use limits to avoid overwhelming users (default 20, max 100)
- For F29 data, ALWAYS use widget tools
